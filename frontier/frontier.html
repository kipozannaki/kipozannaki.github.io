🔭 专题说明

这一专题不追求体系完整，追求跟得上节奏——聚焦"AI 现在最前沿在探索什么"，内容会跟着最新论文和产业动态持续更新，不像 Application / Foundation 两个专题那样有一套稳定不变的知识体系：

  • 推理范式的工程实现　o1 / r1 这类具体推理模型是怎么被训练和部署出来的
  • 算力换智能　Test-Time Compute、Inference Scaling，把"多算一会儿"变成一种可以量化的能力提升手段
  • 搜索与规划　Tree Search、MCTS 在生成式模型里的具体应用
  • 新的训练信号　RLVR（可验证奖励的强化学习）这类正在被验证的对齐新思路
  • 超越单轮问答　World Model、Agent Memory，让模型具备更长时间尺度的状态和推演能力
  • 多模态与协作协议　Multimodal Agent、A2A、MCP Evolution，模型之间、模态之间怎么互相协作

一个需要先说清楚的边界：Foundation 专题里有一篇 Reasoning，讲的是"为什么模型能/不能推理"这种机制层面的问题；这里的 Reasoning Model 讲的是 o1 / r1 这类具体推理范式，工程上到底是怎么被训练和实现出来的——两篇不是同一件事，各自讲各自的层次。

📚 本专题文章

🧠

Reasoning Model

为什么 o1 / r1 这类模型要先"想"很久才回答？为什么想得越久不一定越对？一篇讲清推理模型到底是怎么被训练出来的。
⏱️

Test-Time Compute

为什么同一个模型多问几次投票选答案，正确率能明显提升？一篇讲清"花测试时算力"到底有哪些具体方法，以及为什么不是越多越好。
📈

Inference Scaling

为什么小模型配合更多推理时搜索，有时候能追上大模型？一篇讲清推理时算力和效果提升之间，到底是什么样的定量关系。
🌳

MCTS / Tree Search

为什么下棋 AI 里的搜索算法，被搬来给语言模型做推理？一篇讲清 MCTS 怎么比"多次采样投票"更聪明地花算力。
🎯

RLVR

为什么训练推理模型不用人类偏好打分，改用"答案对不对"当奖励？一篇讲清 RLVR 这套可验证奖励的强化学习到底解决了什么问题，又在哪里会被钻空子。
🌍

World Model

为什么生成一段逼真的视频，不代表模型真的理解物理规律？一篇讲清"世界模型"这个概念在讨论什么，以及为什么这个问题目前还没有定论。
🧷

Agent Memory

为什么带记忆功能的 Agent 有时候会"记错"、引用过时信息？一篇讲清 Agent Memory 到底该记什么、怎么检索、怎么遗忘。
🖇️

Multimodal Agent

为什么模型能"看图说话"，不代表它能成为可用的多模态 Agent？一篇讲清"能看见、能看懂、能行动"之间隔着的那几层工程鸿沟。
🤝

A2A

Agent 之间的通用语言：Google 主导、Linux Foundation 托管的 Agent-to-Agent 协议到底是什么？
🔌

MCP Evolution

月下载 11 亿次的 MCP，为什么 2026 年开始从神器变成 Agent 的瓶颈？
💭

Long Thinking

让 AI 学会该快时快、该慢时慢、该停时停——Test-Time Compute 的终极形态,是智商税还是新边疆？
🪞

Global Workspace

模型嘴上说的和心里真在算的,是同一套表征吗？Anthropic 2026.7 可解释性研究给出了一个反直觉的答案——是,但只占不到十分之一。
