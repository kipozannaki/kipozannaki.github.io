🏠 关于本站

这是我的 AI 主题内容站，围绕三个方向写实战笔记：

  • 应用与编排——怎么把 LLM 用起来：Agent / Skill / Harness / Context / Prompt / Loop 六块拼图
  • 底层与算法——AI 本身是怎么工作的：Tokenization / Embedding 起步，逐步深入 Transformer 与训练范式
  • 前沿研究——AI 现在最前沿在探索什么：推理范式、Test-Time Compute、协作协议

每一篇都是"读 + 写 + 改"的实战笔记——不是把官方文档换个皮，是"我跑一遍踩了哪些坑、最后怎么收的口"。

目标读者：想用 LLM 做点真东西、不想被 PPT 词条忽悠的开发者；刚开始上路不知道怎么做的新手小白。
