🧬 专题说明

这一专题聚焦"AI 本身是怎么工作的"——剥开 Agent 那些花哨的工程包装，回到底层：

  • 模型结构　Transformer / Attention 的设计与变体、Mamba 等新架构的取舍
  • 训练范式　预训练 / SFT / RLHF / DPO 的链路与权衡
  • 推理优化　KV Cache、量化、Speculative Decoding 的工程实践
  • 评测方法　怎么测一个模型"真的"变好，而不是只在几个 Benchmark 上刷分

和 Application 专题不同——这一专题关心的是"为什么模型会这样回答"，而不是"怎么让模型按我的要求回答"。
