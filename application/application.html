📦 专题说明

这一专题聚焦"怎么把 LLM 用起来"——不关心模型本身怎么造(那是 Foundation 专题的事),只关心手上有了一个能对话的模型之后,怎么把它拼成一个真正能交付任务的系统:

  • 怎么写好提示词　Prompt Engineering——同一段话改改顺序、举几个例子,效果就明显不一样
  • 怎么管理上下文　Context Engineering——上下文窗口塞满了怎么办,塞什么才不浪费
  • 怎么让模型查资料　RAG——纯靠模型"记"已经不够了,要让它能现查
  • 怎么让模型用工具　Skill / Function Calling——让模型"伸手"调用外面的 API
  • 怎么让模型持续干一件复杂事　Agent / Workflow——把多步任务拆开,让模型自己来回迭代
  • 怎么约束模型不跑偏　Harness / Loop——给模型套一层"脚手架",让它在工程边界内反复打磨

和 Foundation 专题不同——这一专题关心的是"怎么让模型按我的要求把活干完",而不是"为什么模型会这样回答"。
